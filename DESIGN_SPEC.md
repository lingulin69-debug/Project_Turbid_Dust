## 1. 整體畫面結構

```
┌─────────────────────────────────────────────────────────────┐
│  頂部 HUD 列（fixed, top-right）                              │  ← z-index: 100
│                              [貨幣][登入按鈕][鈴鐺]            │
├──┬──────────────────────────────────────────────────────┬───┤
│  │                                                      │   │
│左│              地  圖  層（可拖曳／縮放）                 │右│  ← z-index: 0
│側│                                                      │側│
│導│              2048 × 1080 px                          │工│
│航│                                                      │具│
│  │                                                      │欄│
│  │                                                      │   │
└──┴──────────────────────────────────────────────────────┴───┘
  ↑ fixed left               面板層（z-index: 60+）               fixed right ↑
```

**螢幕座標系統：** 以瀏覽器視窗為基準，`fixed` 定位

## 2. 地圖層規格

### 2-A. 地圖畫布尺寸

| 屬性 | 數值 | 說明 |
|------|------|------|
| **畫布寬度** | **2048 px** | 設計師繪製底圖的總寬度 |
| **畫布高度** | **1080 px** | 設計師繪製底圖的總高度 |

> 地圖為單一完整底圖，陣營區域劃分由程式邏輯處理，設計師無需標註分界。

### 2-B. 地圖交互行為（設計師參考）

| 行為 | 說明 |
|------|------|
| 拖曳 | 使用者可用滑鼠/手指拖曳瀏覽 |
| 縮放範圍 | 0.5x ～ 3.0x，初始 1.0x |
| 縮放基準點 | 畫布中心 `center center` |
| 未登入效果 | 底圖套 `blur(4px)` + `grayscale(100%)` |
| 戰爭迷霧 | 敵方半側套 `rgba(0,0,0,0.6)` + `backdrop-blur-sm`（程式處理，設計師不需製作） |

### 2-C. 地圖底圖素材需求

| 素材 | 尺寸 | 格式建議 | 備註 |
|------|------|---------|------|
| **地圖底圖** | **2048 × 1080 px** | PNG / WebP | 單一完整底圖 |

## 4. 面板視窗規格（WhiteCrowCard）

> 用於：公告、任務、日誌、圖鑑、背包、NPC、惡政等所有主要面板

| 屬性 | 數值 |
|------|------|
| **最大寬度** | **450 px** |
| 手機寬度 | `100vw - 32px`（螢幕寬度 - 左右各 16px 邊距）|
| **最小高度** | **480 px**（桌機，維持畫框感）|
| 最大高度 | `100vh - 80px` |
| **圓角** | **8 px** |
| 內邊距（桌機）| 上下 32px，左右 24px |
| 內邊距（手機）| 上下 24px，左右 16px |

### 面板內部結構（從上到下）：

```
┌────────────────────────────────────┐  ← 面板外框 450×480px min
│ ── 裝飾線 ──────────────────────── │  h: 1px，漸層
│  [ CODE LABEL ]                    │  h: ~24px（含下方 margin 16px）
│  ◇ ─── 面板標題 ─── ◇             │  h: ~40px
│ ── 裝飾線 ──────────────────────── │  h: 1px，漸層
│  ─────────────────────────────── │  border-bottom（header 底線）
│                                    │
│   [ Tab 標籤列（若有）]              │  h: ~50px（含 padding top 16px bottom 16px）
│   ─────────────────────────────   │
│                                    │
│   內容區（可捲動）                   │  flex: 1（剩餘高度）
│   ... 文字 / 格狀物品 / 列表 ...    │
│                                    │
│   Node_Authorization_Granted       │  h: ~32px（含 padding top 24px）裝飾文字
│ ── 導航圓點（若多頁）─────────────  │  h: ~34px（含 padding top 24px）
└────────────────────────────────────┘
                          ↑ 關閉按鈕（絕對定位，右上角 36×36px）
```

### Tab 列規格：

| 屬性 | 數值 |
|------|------|
| Tab 高度 | 上下 padding 各 16px + 文字 11px |
| 字體 | Noto Sans TC，11px，letter-spacing 0.2em |
| 啟用指示 | 底部 2px 色條 + 頂部 4px 圓點 |

## 10. 音效素材清單

> 音效檔放置於 `/public/sounds/`，程式讀取路徑固定，設計師/音效師提供後直接替換即可。

| 檔名 | 觸發時機 | 建議音效風格 | 時長建議 |
|------|---------|------------|---------|
| `page_flip.mp3` | 公告 / 任務 / 日誌 / 圖鑑 面板開啟 | 紙質翻頁聲，輕柔 | 0.3–0.5 秒 |
| `bell.mp3` | 頂部鈴鐺按鈕點擊（通知） | 清脆鈴鐺聲 | 0.5–1.0 秒 |
| `coin.mp3` | 貨幣欄位點擊 | 金屬錢幣碰撞聲 | 0.3–0.6 秒 |

> **格式**：MP3（相容性最佳）；建議同時提供 OGG 備用
> **音量**：母帶音量建議 -12 dBFS，程式會自行調整各音效音量
