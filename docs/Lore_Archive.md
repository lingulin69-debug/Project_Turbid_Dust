# 《白鴉之繭》世界觀與故事設定 (Lore Archive)

## 1. 創世悲劇：星隕之日的誤解

- **起源**：一對雙生兄弟觸碰了天外落下的「星隕之繭」。

- **弟弟（深淵之主）**
  - **狀態**：墜入大地裂痕，與原始黑泥融合，成為蔓延世界的「黑霧」源頭。
  - **動機重構**：其行動並非單純營救，而是帶著**「被拋棄感」**的質問。他想知道，為何在那道光芒落下時，哥哥沒有拉住他，而是讓他獨自墜入深淵。

- **哥哥（繭的核心）**
  - **狀態**：肉體晶體化，囚禁在聖座深處。
  - **意識對立**：他的痛苦中夾雜著對弟弟**「獲得自由」**的怨恨。相關對話應展現「冷漠、混亂、指責」的潛台詞——他認為弟弟在外面享受自由，而讓他獨自承受繭的折磨。

## 2. 兩大陣營立場

### 【濁息者 Turbid-Breathers】
- **視覺色調**：深紫黑 `#211a2e` / 紫色高光 `#7c3aed`
- **現狀**：被黑霧感染後存活的異類。皮膚出現礦石裂紋，擁有異能。
- **信仰**：追求殘酷的真實，認為黑霧是弟弟在呼喚哥哥，是世界的另一種可能。

### 【淨塵者 Dust-Cleaners】
- **視覺色調**：米白金 `#ede6ce` / 白金高光 `#d4af37`
- **現狀**：靈魂殘缺，天生情感淡漠。因這種「空洞」而對黑霧免疫。
- **信仰**：守護虛假的和平，依賴教會血清維持人性。
- **潛在危機**：若血清中斷，會迅速「白瓷化」成為失恩者。

> ⚠️ 舊版文件中淨塵者色調為「深綠/古銅」，已更正為上方色碼，以代碼為準。

## 3. 結局分支

- **物理結局**：章節結算時，根據天平偏移決定誰勝出。

- **真結局【灰色時代】Era of Gray**（P3，長期規劃）
  - **解鎖核心**：必須解開「星隕之日」雙方的誤解。
  - **條件**：玩家需收集齊所有「昔日的餘溫」系列碎片，解鎖最終隱藏文案，揭示當時斷裂的繩索並非弟弟主動放手。
  - **結局表現**：不再是黑或白的勝出，而是光與影的完美和解，世界回歸混沌而平靜的灰色。

## 4. 第一章：餘燼的呼吸 (Chapter 1)

### 核心據點
- **遺棄醫院**：發現被遺忘的白瓷化初期實驗體。
- **邊境哨所**：目睹教會強制回收「失效」的藥劑瓶。
- **無名墳塚**：獲得哥哥遺留的斷裂木笛（遺物）。

### 章節分支抉擇
你要選擇「慈悲的安樂死」還是「痛苦地尋求真相」？

### 因果標籤範例（第一章）

## 5. 關鍵道具與遺物

### 斷裂的木笛 The Broken Flute
- 弟弟送給哥哥的最後禮物。
- 於無名墳塚可獲得。
- 集齊「昔日的餘溫」三件遺物後，解鎖展示繭降臨前夕真相的無聲動畫。
- 識別碼：`relic_broken_flute`

## 6. 寫作風格指引

生成任務文本、NPC對話、小道消息時遵守：

```
語氣：沉重、詩意、不過分說明
節奏：短句優先，長句用於情緒高點

禁止：
  - 過度解釋隱喻
  - 提前劇透雙生子真相
  - 輕描淡寫白瓷化的恐懼感

鼓勵：
  - 感官細節（氣味、聲音、觸感）
  - 模糊的對話（說話者不明）
  - 暗示多於直述
```
