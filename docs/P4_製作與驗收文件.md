## 1. 旅店救援距離功能

### 功能說明

玩家失蹤後，其他玩家可至旅店委託救援（5 幣）。
旅店老闆根據**旅店所在據點**與**失蹤玩家所在據點**的距離，提供不同等級的時間縮短效果。

### 距離計算方式（全自動，不需手動填表）

系統根據**地圖座標**自動計算距離：
- 每個據點都有 x/y 座標（0–100 的百分比數值）
- 距離 = √((旅店x - 玩家x)² + (旅店y - 玩家y)²)
- 比對門檻值，決定縮短等級

| 等級 | 距離條件 | 縮短失蹤時間 |
|------|---------|-------------|
| 近區 | 距離 < 25 | **60 分鐘** |
| 中區 | 距離 25–50 | **30 分鐘** |
| 遠區 | 距離 > 50 | **10 分鐘** |

收費：固定 **5 幣**（無論距離等級）

### rescue-distance.json 資料格式（已填寫）

檔案路徑：`src/data/rescue-distance.json`

```json
{
  "near_threshold": 25,
  "medium_threshold": 50,
  "time_reductions": { "near": 60, "medium": 30, "far": 10 },
  "rescue_cost": 5
}
```

> 若需調整門檻，修改 `near_threshold` / `medium_threshold` 數值即可。
> 數值越小 = 近區範圍越窄；數值越大 = 近區範圍越寬。

### 待完成項目

#### 1-A. ✅ rescue-distance.json 資料格式已確認
- 採用**座標自動計算**方式，不需手動列出每個據點
- 已填入門檻數值（near: 25, medium: 50）
- 若要調整遠近範圍，修改門檻數值即可

#### 1-B. ✅ 後端 API 實作（已完成）
- `GET /api/inn/rescue-distance` — 回傳設定檔（已更新為新格式）
- `POST /api/npc/inn/rescue` — 距離計算邏輯已改寫：
  1. 讀取旅店的 `current_landmark_id` → 查詢 `landmark-positions.json` 取得座標
  2. 讀取失蹤玩家的 `current_landmark_id` → 同檔取得座標
  3. 計算歐幾里得距離 √(Δx² + Δy²)
  4. 比對門檻值決定縮短分鐘數（10 / 30 / 60）
  5. 扣費從 hardcode 5 改為讀取 `rescue_cost` 設定值
- 新增 `src/data/landmark-positions.json` — 後端唯一座標來源
  1. 取得失蹤玩家的 `current_landmark_id`
  2. 取得旅店老闆的 `current_landmark_id`
  3. 比對 rescue-distance.json，找出對應等級
  4. 套用對應的時間縮短效果

#### 1-C. 前端顯示
- 旅店 NPC 互動 Modal 的「委託救援」區塊
- 輸入目標 OC 名稱後，顯示預估縮短時間（查表顯示，讓玩家先知道效果）
- 格式：`「預計縮短失蹤時間：XX 分鐘」`

### API 驗收標準（1-B）
- 救援後失蹤時間依距離等級正確縮短
- 若找不到距離設定，預設縮短 10 分鐘
- 費用扣除正確（5 幣）
- 餘額不足時拒絕請求，回傳 `INSUFFICIENT_COINS`

## 2. NPC 互動視窗背景圖

### 功能說明

每個 NPC 的互動視窗有一張**全版底圖**，包含：
- 視窗底色/背景紋理
- 角色立繪（與背景合為一張圖，不分開）

整張圖直接作為視窗的 `background-image` 或 `<img>` 貼入。

### 美術規格

| 項目 | 規格 |
|------|------|
| 視窗最大寬度 | 420px |
| 建議圖片寬度 | **840px**（@2x，高解析） |
| 建議圖片高度 | 自訂（依內容高度，建議 600–900px） |
| 格式 | PNG（含透明度）或 JPG |
| 立繪位置 | 圖片右側或左側，與底圖融合 |

### 檔案命名規則（放入 `public/assets/npc-bg/`）

| 檔名 | NPC |
|------|-----|
| `npc_black_merchant_bg.png` | 🎭 黑心商人 |
| `npc_item_merchant_bg.png` | 📦 道具商人 |
| `npc_inn_owner_bg.png` | 🏠 旅店老闆 |
| `npc_pet_merchant_bg.png` | 🐾 寵物商人 |
| `npc_trafficker_bg.png` | 🪤 人販子 |

### 驗收標準（2-B）

- 底圖不影響 UI 元件的可操作性
