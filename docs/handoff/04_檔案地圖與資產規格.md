## 8. API 端點總覽

### P0 — 核心基礎
| Method | Path | 說明 |
|--------|------|------|
| POST | `/api/auth/login` | Email/密碼登入 |
| POST | `/api/auth/register` | 註冊 |
| GET | `/api/profile/:id` | 取得玩家檔案 |
| PUT | `/api/profile/:id/faction` | 選擇陣營 |
| POST | `/api/identity/select` | 選擇身份（公民/叛教者/領導者） |

### P1 — 地圖 / 地標 / 天平
| Method | Path | 說明 |
|--------|------|------|
| GET | `/api/landmarks` | 取得所有地標 |
| GET | `/api/landmarks/:id` | 單一地標詳情 |
| GET | `/api/balance` | 取得天平狀態 |
| POST | `/api/balance/contribute` | 投入貨幣影響天平 |

### P2 — NPC / 組隊 / 骰子
| Method | Path | 說明 |
|--------|------|------|
| POST | `/api/party/create` | 建立隊伍 |
| POST | `/api/party/join` | 加入隊伍 |
| GET | `/api/npc/:type` | NPC 互動（5 類型） |
| POST | `/api/dice/roll` | 擲骰子 |

### P3 — 叛教者 / 清算者 / 綁架
| Method | Path | 說明 |
|--------|------|------|
| POST | `/api/apostate/activate` | 啟動叛教者身份 |
| POST | `/api/apostate/action` | 叛教者行動（greed/leak/pickpocket） |
| POST | `/api/liquidator/activate` | 啟動清算者 |
| POST | `/api/kidnap/initiate` | 發起綁架 |

### P4 — 結算 / 賽季
| Method | Path | 說明 |
|--------|------|------|
| POST | `/api/settlement/trigger` | 觸發天平結算 |
| GET | `/api/leaderboard` | 排行榜 |
| GET | `/api/season/current` | 當前賽季 |

### P5 — 章節轉場
| Method | Path | 說明 |
|--------|------|------|
| POST | `/api/chapter/advance` | 推進章節（需 ADMIN_PASSWORD）|
| GET | `/api/chapter/current` | 取得當前章節與上次結算 |

## 9. 美術素材規格

### 命名規則
```
肖像：  landmark_l{章}_t{編號}.png / landmark_l{章}_p{編號}.png
NPC：   {npc_type}.png（如 black_merchant.png）
地標：  landmark_{id}.png
圖示：  icon_{name}_{size}.png
```

### 尺寸規格

| 素材類型 | 原始尺寸 | 顯示尺寸 | 格式 |
|----------|---------|---------|------|
| 角色肖像 | 1120×1600 @2x | 560×800 | PNG（透明背景） |
| 地圖背景 | 2048×1080 | 響應式 | PNG / WebP |
| NPC 背景 | max-width 420px | 自適應 | PNG |
| 圖示 (小) | 128×128 | 64×64 | SVG / PNG |
| 圖示 (大) | 256×256 | 128×128 | SVG / PNG |
| UI 按鈕 | 按需 | 按需 | SVG 優先 |
| Lottie 動畫 | — | — | JSON |

### 色彩規格
```
淨塵者 (Pure/Dust-Cleaners):
  主色: #ede6ce（米白金）
  高光: #d4af37（白金）

  高光: #7c3aed（紫色）
  文字: #e0d6ff

共用:
```
