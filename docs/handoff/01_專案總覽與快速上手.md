## 1. 這是什麼專案？

**《白鴉之繭》(Project Turbid Dust)** 是一個以繪圖企劃為核心的網頁輔助遊戲平台。

- **企劃性質**：繪圖企劃 + 網頁輔助玩法（非純遊戲）
- **玩家人數**：40 人，分兩陣營各 20 人
- **企劃長度**：兩個月
- **NPC**：另計，無陣營歸屬，由真人扮演
- **更新節奏**：前期每週一章，後期每 10 天一章
- **核心玩法**：陣營貨幣爭鬥 → 影響中央天平 → 決定世界走向

## 2. 技術棧

### 目前使用中（雙軌並行）

| 層級 | 技術 | 備註 |
|------|------|------|
| **遊戲引擎** | Cocos Creator 3.x | TypeScript，主力開發方向 |
| **Web 前端** | React 18 + Vite + TypeScript | 原產品，仍在運行 |
| **CSS 框架** | Tailwind CSS | 含自訂主題（PTD_UI_Theme） |
| **動畫** | Framer Motion（Web）/ cc.Tween（Cocos） | Web 版正在遷移 |
| **後端** | Express.js（Node.js） | `server/index.ts`，Port 3001 |
| **主資料庫** | Supabase (PostgreSQL) | 即時同步天平、用戶資料 |
| **輔助資料庫** | SQLite (Prisma 7) | 身分選拔、本地數據 |
| **API 通訊** | REST API | `src/api/client.ts` + `VITE_API_URL` 環境變數 |

### 關鍵依賴

```
前端：react, react-dom, @supabase/supabase-js, framer-motion,
      @use-gesture/react, lottie-react, tailwindcss
後端：express, @prisma/client, cors, node-cron
工具：vite, typescript, eslint, postcss
```

## 3. 如何啟動

### Web 版（React + Vite）

```bash
# 1. 安裝依賴
npm install

# 2. 環境變數（複製 .env.example 為 .env.local）
cp .env.example .env.local
# 填入 Supabase URL、Anon Key、VITE_API_URL

# 3. 啟動前端
npm run dev

# 4. 啟動後端（另開終端）
npx ts-node server/index.ts
# 後端跑在 http://localhost:3001
```

### Cocos Creator 版

```
1. 用 Cocos Creator 3.x 開啟專案根目錄
2. 使用內建預覽功能
3. 場景入口：assets/Scene/MapTestView.scene
```

## 5. 核心設計原則（必須牢記）

1. **天平只被貨幣影響** — 任何機制不得直接修改天平值
2. **背道者身分不是要被抓住的** — 遊戲樂趣在貨幣與天平的陣營爭鬥
3. **先修復閉環缺口，再開發新功能**
4. **靜態文本存 `/src/data/`** — 不硬編碼在組件裡
